h1. etikett 

etikett helps you to render uris. It provides icons, labels and shortnames for each uri. The etikett database is created on startup from skos files or similar rdf. etikett can be used to share display information between applications. Etikett helps you to create and maintain json-ld context documents. This can be extremely useful if you want to create json files dynamically from rdf data.   

Comes with the following testsets

h1. Requirements

1. Java 8
2. Typesafe Activator

bc.. cd /tmp
wget http://downloads.typesafe.com/typesafe-activator/1.3.2/typesafe-activator-1.3.2-minimal.zip
unzip typesafe-activator-1.3.2-minimal.zip
sudo mv activator-1.3.2-minimal /opt

h1. Run

h2. Download

bc.. cd /tmp
git clone https://github.com/hbz/etikett
cd etikett

h2. Run

This will start the application in developer mode. Some test data is loaded at startup

bc.. /opt/activator-1.3.2-minimal/activator run

p. Go to http://localhost:9000

h2. Manual Test

List all

bc.. curl "http://localhost:9000" -H"accept: application/json"

p. Or list info for a single uri

bc.. curl "http://localhost:9000/?url=http%3A%2F%2Fpurl.orms%2Fissued" -H"accept: application/json"

h1. Install on Ubuntu

bc.. 
cd /tmp/etikett
/opt/activator-1.3.2-minimal/activator dist
cp target/universal/etikett-0.1.0-SNAPSHOT.zip  /tmp
cd /tmp
unzip etikett-0.1.0-SNAPSHOT.zip
mv etikett-0.1.0-SNAPSHOT /opt/etikett

p. edit startscript

bc.. sudo cp /tmp/etikett/install/etikett.tmpl /etc/init.d/etikett
sudo chmod u+x /etc/init.d/etikett
sudo editor /etc/init.d/etikett

p. set the following vars

bc.. JAVA_HOME=/opt/java
HOME="/opt/etikett"
USER="user to run etikett"
GROUP="user to run etikett"
SECRET=`uuidgen` # generate a secret e.g. using uuidgen
PORT=9000

p. include into system start and shutdown

bc.. sudo update-rc.d etikett defaults 99 20

p. Create/Configure Mysql Table

bc.. mysql -u root -p
CREATE DATABASE etikett;
CREATE USER 'etikett'@'localhost' IDENTIFIED BY 'etikett';
GRANT ALL ON etikett.* TO 'etikett'@'localhost';

p. Set /opt/etikett/conf/application.conf

bc.. db.default.driver=com.mysql.jdbc.Driver
db.default.url="jdbc:mysql://localhost/etikett"
db.default.user=etikett
db.default.password="etikett"

p. start

bc.. sudo service etikett start

rm -rf /opt/etikett/*

sudo service etikett start
